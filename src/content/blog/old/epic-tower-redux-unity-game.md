## Epic Tower Redux - Build Your Sky-High Dreams

Epic Tower Redux is a physics-based mobile game that challenges players to build the tallest tower possible by strategically dropping boxes. The game combines precision timing with physics simulation to create an engaging and addictive gameplay experience.

### Core Gameplay

- **Box Dropping Mechanics**: Precise timing and positioning of falling boxes
- **Physics Simulation**: Realistic tower stability and collision physics
- **Height Challenges**: Progressive difficulty as towers grow taller
- **Score Competition**: Global leaderboards for the highest towers

### Technical Achievements

Developed using **Unity** with **C#**, the game features:
- **Advanced Physics**: Realistic box physics and tower stability calculations
- **Performance Optimization**: Smooth gameplay across various mobile devices
- **Cross-platform Development**: Simultaneous iOS and Android deployment
- **Visual Effects**: Particle systems and visual feedback for enhanced user experience

### Development Challenges

This project involved solving several interesting technical problems:
- **Physics Optimization**: Balancing realistic physics with mobile performance constraints
- **Touch Controls**: Implementing intuitive touch-based box dropping mechanics
- **Dynamic Difficulty**: Creating engaging progression without frustrating players
- **Memory Management**: Optimizing for mobile device limitations

### Unity Implementation Highlights

- **Rigidbody Physics**: Leveraging Unity's physics engine for realistic box behavior
- **Mobile Input Systems**: Custom touch gesture recognition for precise control
- **Dynamic Object Management**: Efficient spawning and cleanup of game objects
- **Cross-platform Optimization**: Platform-specific performance tuning

### Player Experience Focus

The game design emphasizes:
- **Accessibility**: Easy to learn, difficult to master gameplay
- **Replayability**: Each attempt offers unique physics-based challenges
- **Visual Satisfaction**: Satisfying visual feedback for successful tower building
- **Mobile-First**: Designed specifically for mobile gaming sessions

**Technology Stack**: Unity Engine, C#, Mobile Physics Systems  
**Platforms**: iOS App Store, Google Play Store  
**Genre**: Puzzle, Physics, Casual Gaming

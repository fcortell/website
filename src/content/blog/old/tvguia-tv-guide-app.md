## TVGuía - Your Real-time TV Guide

TVGuía is an iOS and Android app that provides real-time television programming information. This project represents my exploration into cross-platform mobile development using React Native with JavaScript.

### Key Features

- **Real-time programming**: Access current and upcoming TV show schedules
- **Multi-country support**: Currently available in Spain, Germany, and USA
- **Cross-platform**: Native experience on both iOS and Android devices
- **User-friendly interface**: Clean, intuitive design for easy navigation

### Technical Implementation

The app was developed using **React Native** with **JavaScript**, allowing for:
- Single codebase deployment across multiple platforms
- Native performance on both iOS and Android
- Efficient development cycle and maintenance

### Availability

TVGuía is designed to be a quick, simple, and free solution for users who want to stay updated with their television programming without the complexity of traditional TV guide apps.

**Platforms**: iOS App Store, Google Play Store  
**Regions**: Spain, Germany, USA  
**Technology Stack**: React Native, JavaScript
